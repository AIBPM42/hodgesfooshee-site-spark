## FAQ

### How much does this cost to run?

**Hosting:** Vercel (free tier or ~$20/month for Pro)

**Database:** Supabase (free tier up to 500MB, then ~$25/month for Pro)

**API costs:**
- Realtyna MLS: Depends on your MLS agreement (typically bundled with MLS access)
- Perplexity AI: ~$0.20 per county refresh (refresh 9 counties monthly = $2/month)
- FAL.ai (Image Studio): ~$0.10 per image (50 images/month = $5/month)

**Total estimated monthly cost: $50-75/month** (less than one Zillow lead)

### How long does setup take?

**Technical deployment:** Already done (system is built)

**Your setup:** 2-3 hours
- Configure API keys
- Set Lead Hunter automation schedule
- Upload initial training documents to AI Q&A
- Review and approve agent accounts

**Agent training:** 15-30 minutes per agent (one orientation session)

### What if an agent leaves? Do they keep access?

No. When you deactivate an agent in the admin dashboard, their access is immediately revoked. They can no longer log in.

You can also set their status to "suspended" (temporary access removal) or "inactive" (permanent removal).

Their assigned leads stay in the system—you can reassign to another agent.

### Can I customize the county intelligence content?

Yes, two ways:

**1. Edit manually:** Go to admin → counties → click county → edit sections

**2. Refresh via AI:** Click "Refresh County" to regenerate content (uses latest data)

You control the tagline, hero image, and featured content.

### How secure is the lead contact information?

**Row-Level Security (RLS):** Only authorized users (brokers, assigned agents) can see lead contact info. Public users see featured leads only (and only if you enable that).

**Encryption:** Database is encrypted at rest and in transit (industry-standard SSL/TLS).

**Audit logs:** Every access is logged (who viewed what, when).

**Compliance:** Designed to meet real estate industry standards for client data protection.

### What happens if the MLS feed goes down?

Your website still works—it serves cached property data. Once the MLS feed is back online, it automatically syncs new/updated listings.

**Fallback:** Properties stay visible until MLS sync resumes. No downtime for visitors.

### Can I add more agents later?

Yes. No limit.

**Process:**
1. Agent applies via registration page
2. You review in admin → agents → pending
3. Approve → they get agent access
4. Reject → they're notified, no access

Takes 2 minutes per agent.

### What if I want to change the design or branding?

The design system is built with customization in mind:
- Color palette (Linen 75) can be swapped for your brand colors
- Logo, fonts, imagery are all configurable
- No code required for basic changes (colors, images, text)
- Developer required for structural layout changes

### How do I know if the AI-generated county content is accurate?

**Perplexity AI** cites sources (it pulls from news, census data, real estate databases, school rating sites, etc.). The content is research-backed, not hallucinated.

**You review before publishing:** When you refresh a county, preview the content. If something looks off, edit it manually.

**Best practice:** Refresh county content quarterly, review for accuracy, then publish.

### Can I export leads or reports?

**Currently:** You can view leads in the dashboard, export via browser (copy/paste to spreadsheet).

**Planned:** CSV export for leads, agents, transactions (Analytics dashboard, coming soon).

### What if I want to sell this system or white-label it for other brokerages?

The system is designed to be white-label ready. You could:
- License it to other non-competing brokerages (different markets)
- Sell it as a turnkey solution
- Use it as a recruiting differentiator (exclusive to your brokerage)

**Your call.** The codebase is yours.
